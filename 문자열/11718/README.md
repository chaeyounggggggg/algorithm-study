## 회고
BufferedReader는 입력을 버퍼에 모아서 읽는 입력방식<br>
readLine()은 한 줄 단위로 문자열을 읽는 메서드로 /n까지 읽어 문자열로 반환하고 더이상 읽을 입력이 없으면 null을 반환한다 <br>
호출될 때마다 입력 한 줄을 소비한다는 특징이 있다. 읽은 값은 변수에 저장해서 재사용하자!!!!!
