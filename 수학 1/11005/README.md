## 회고
StringBuilder를 사용하면 더 편해진다<br>
StringBuilder는 문자열을 계속 바꿔도 새 객체를 만들지 않는 문자열 도구이다.<br>
String은 매번 새 문자열 객체를 생성하지만 StringBuilder는 내부 버퍼에 붙이기만 해서 속도가 빠르고 메모리 낭비가 없다.<br>
StringBuilder에 reverse함수도 있어서 뒤집어 출력할때도 편하다.
